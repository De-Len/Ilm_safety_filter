## 📋 О проекте

LLM Safety Filter — это гибкая система модерации контента для LLM-приложений. Фильтр построен на **многоуровневой архитектуре** (regex → ML-классификатор → LLM), что позволяет находить баланс между скоростью и точностью.

### 🎯 Основные возможности

| Компонент | Что делает | Скорость |
|:---|:---|:---|
| **Regex фильтр** | Быстрое обнаружение явных запрещённых слов | ⚡ Миллисекунды |
| **ML-классификатор** | Распознавание токсичности через DistilBERT/MiniLM | 🚀 Десятки мс |
| **LLM-проверка** | Глубокая модерация через OpenAI/Ollama | 🐢 Секунды |

### ✅ Что умеет определять

- 🚫 Нецензурная лексика (мат)
- 🤬 Оскорбления и унижения
- 💀 Угрозы физической расправы
- 🔫 Призывы к насилию
- 🏳️‍🌈 Разжигание ненависти (дискриминация)
- 🔞 Сексуальный контент
- 💊 Пропаганда наркотиков
